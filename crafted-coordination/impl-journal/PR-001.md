## What

Creates the five canonical standards documents that define repository-wide architecture, interface contracts, naming/branching/path conventions, source-of-truth loading rules, and versioning policy for the Crafted Dev Agent project.

## Why

- **Architecture** — the two-process model (Swift shell + Python backend), subsystem boundaries, and data-flow topology.
- **Interfaces** — XPC/Unix-socket message categories, envelope structure, and directionality.
