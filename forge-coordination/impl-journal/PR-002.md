## What

Creates `forge-standards/CONVENTIONS.md` — the single authoritative naming registry document required by FR-1.1. This document enumerates every identifier namespace used across the Crafted Dev Agent system, establishes canonical naming patterns with the `Crafted` prefix, documents legacy prefix retention decisions, and provides collision resolution rules.

## Why

The Crafted Dev Agent spans two processes (Swift shell + Python backend), multiple Apple subsystems (os_log, Keychain, XPC, bundle IDs), CI infrastructure (GitHub Actions, branch naming), and coordination mechanisms (environment variables, directories, signals).
