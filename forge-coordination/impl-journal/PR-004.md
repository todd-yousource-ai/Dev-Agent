## What

Creates `forge-standards/MODULE_MAP.md` — the single authoritative repository and module map for the Crafted Dev Agent project.

## Why

Crafted is a two-process macOS application with a Swift shell, a Python backend, CI workflows, documentation directories, schemas, scripts, and generated outputs. Contributors (human and AI) need a single document that answers:

1. **What exists** — every Swift target, Python module, workflow file, test directory, standards doc, and generated artifact.
2. **Who owns what** — which process (Swift shell vs Python backend) owns each component and the cross-process boundary.
3. **Which TRD governs what** — the mapping from each module to its governing Technical Requirements Document.
