## What

Establish the single source-of-truth versioning system for Consensus Dev Agent by creating:

1. **`VERSION`** — A plain-text file at repository root containing the canonical semantic version (`0.1.0`)
2. **`docs/VERSIONING.md`** — The complete versioning contract specification defining how the version propagates across the Swift shell, Python backend, CI artifacts, and the inter-process handshake protocol

## Why

Consensus Dev Agent is a two-process architecture (Swift shell + Python backend) that communicates via an authenticated Unix socket. Version coherence across these boundaries is a **correctness requirement**, not just a convention.
